# bootstrap.sh #

bootstrap on macOS

## Firefox ##

### separate search engine in private browsing mode ###

1. [about:config](about:config)
2. set **browser.search.separatePrivateDefault** **true**
3. [about:preferences#search](about:preferences#search)
